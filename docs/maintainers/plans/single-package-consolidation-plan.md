# Single-Package Consolidation — Superseded

The multi-distribution structure (`coactra-ai`, `coactra-memory`, etc.) has been
consolidated into a single `coactra` package as of the alpha redesign. This plan
document describes work that is now complete.

The current package and rename direction is documented in the system vision and
rename migration specs:

**[design/2026-06-06-coactra-vision.md](https://github.com/DataOpsFusion/coactra/blob/main/design/2026-06-06-coactra-vision.md)**

**[design/2026-06-06-rename-migration.md](https://github.com/DataOpsFusion/coactra/blob/main/design/2026-06-06-rename-migration.md)**

The next rename milestone is `coactra.jobs` / `coactra.directory` →
`coactra.workflow` / `coactra.team` (mechanics in the rename migration spec above).

# Research Workspace

Create a scoped workspace with notes, reports, a handoff note, and a capability
manifest. Local command execution is not enabled.

```bash
python3 examples/projects/research_workspace/app.py
```

Shows:

- `open_workspace(...)`
- scoped file writes and reads
- handoff notes
- passive capability manifests

Source: [examples/projects/research_workspace](https://github.com/DataOpsFusion/coactra/tree/main/examples/projects/research_workspace)

# Release Checkpoint

A project-shaped release ledger. It tracks release progress with durable work
checkpoints and emits an artifact at completion.

```bash
python3 examples/projects/release_checkpoint/app.py
```

Use this as the starting point for release automation, publish jobs, or other
long-running operational tasks.

Source: [examples/projects/release_checkpoint](https://github.com/DataOpsFusion/coactra/tree/main/examples/projects/release_checkpoint)

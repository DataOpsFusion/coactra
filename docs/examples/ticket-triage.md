# Ticket Triage

A small project that combines agent drafting, resolution memory, durable work,
checkpoints, artifacts, and events.

```bash
python3 examples/projects/ticket_triage/app.py
```

This is the best copy-and-modify project example. Replace only the backend
boundaries for production: memory backend, work store, and AI port.

Source: [examples/projects/ticket_triage](https://github.com/DataOpsFusion/coactra/tree/main/examples/projects/ticket_triage)
